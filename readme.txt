This is Heimerdinge, a Discord bot designed to display summoner information for the game League of Legends. I have attached some useful information below if the project is to be replicated!
Running Python version 3.12.4
Python must be running the package requests in order to make api calls, can be done by typing pip install requests into cmd
I have used this website https://embed.dan.onl/ in order to streamline the menu creation process, cool tool!
All assets can be downloaded via https://developer.riotgames.com/docs/lol
